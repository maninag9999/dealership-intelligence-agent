# Dealership Intelligence Agent

A multi-modal agentic analytics system for auto dealership operations. Combines text-to-SQL, semantic review search, and on-demand ML inference behind a conversational LangGraph agent.

## Status

🚧 Under active development. Week 1 in progress (data foundation).

## Tech Stack

- **Data:** DuckDB, dbt, Great Expectations, Polars
- **ML:** XGBoost, DistilBERT, scikit-learn, MLflow
- **Agents:** LangGraph, Ollama (Llama 3), ChromaDB
- **Serving:** FastAPI, Streamlit

All components run locally — no paid APIs.
